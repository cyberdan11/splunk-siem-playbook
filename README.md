# splunk-siem-playbook

## Objective
Demonstrate hands-on SIEM skills using Splunk:
- Log ingestion
- Detection engineering
- Security monitoring

## Environment
- Splunk Enterprise (local VM)
- Windows 10 + Sysmon
- Apache web server logs

## Contents
- `detections/` – Splunk searches and Sigma rules
- `logs/` – Sample log data used for detections
- `screenshots/` – Proof of ingestion and detections firing
- `lab-setup/` – Step-by-step environment setup

## Key Skills Demonstrated
- SPL (Search Processing Language)
- Log normalization
- Security detection logic
- Incident visibility
